Privacy Policy

Sawa Notifications is a Chrome extension designed to display notifications to authenticated users based on messages received from the Sawa CRM system.

Authentication

The extension uses a secure Single Sign-On (SSO) process to authenticate users through the Sawa CRM website (https://maxim.bedab.com) and obtain an access token. This token is used solely to retrieve notification messages.

Information Collection and Use

The extension does not collect personal information directly from users. Any data accessed from the CRM system is used only to display notifications within the browser.

Data Storage

Authentication tokens and notification-related data may be stored temporarily using Chrome storage to maintain the user session and functionality. No personal data is permanently stored.

Permissions

The extension uses the following permissions:

  • notifications – to display CRM notifications
  • storage – to store session-related data
  • alarms – to periodically check for new notifications

Data Sharing

No user data is sold, shared, or transferred to third parties. All data communication occurs only between the extension and the Sawa CRM system.

User Consent

By installing and using this extension, users consent to authentication via the Sawa CRM system and receiving notifications generated by it.
